
# 2025 Team SH35 Project

## Setup

### ❗Download Docker 
1. Before you can do anything witht this progject you will need to install docker. [Windows Version](https://docs.docker.com/desktop/setup/install/windows-install/) or [Mac Version](https://docs.docker.com/desktop/setup/install/mac-install/)
2. If on Mac or Windows you will need to run Docker Desktop to start the Docker server before you can use any docker commands

### Running the Backend and  Frontend
1. In terminal navigate project's root directory 
2. Run the command `docker compose up`

### Running the database 
1. Navigate to the `database` directory in terminal
2. Run the command `docker compose up`
